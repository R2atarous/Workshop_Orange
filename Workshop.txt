Library ที่คาดว่าจะใช้สำหรับ Project นี้ พร้อมทั้งระบุด้วยว่าทำไมถึงเลือกใช้
1. lodash สำหรับจัดการข้อมูล
2. react-router สำหรับจัดการ route หน้า
3. joi สำหรับการทำ Validate
4. moment สำหรับจัดการ date
5. react-redux สำหรับจัดการกับ stage ของข้อมูล

จะต้องจัดการ State ของข้อมูลอย่างไร เพื่อรองรับกับโจทย์นี้ได้
ใช้ react-redux มาจัดการกับ state ของข้อมูล โดย store จะเก็บ state ทั้งหมดไว้ และถ้าทำการแก้ไข state ต้องทำ action เมื่อมี action กับ state แล้ว reducers จะทำการ return state ใหม่กลับมา

จะใช้วิธีใดในการ Validate Form สำหรับการ Save Draft และสำหรับการ Submit
Validate Form จะเริ่มทำงาน เมื่อมีการกรอกข้อมูลมากกว่า 1 field เป็นต้นไป สำหรับปุ่มบันทึก (Save Draft) ถึงจะสามารถใช้งานได้ แต่ถ้าไม่มีการกรอกข้อมูลเลย ปุ่มบันทึก (Save Draft) 
จะทำการ disable ไม่สามารถใช้งานได้ และปุ่มหน้าต่อไปกับปุ่มส่งข้อมูลสำหรับการ Submit จะทำการ Validate Form เมื่อกรอกข้อมูลครบทุกช่องถึงจะสามารถใช้งานปุ่มได้
แต่ถ้ากรอกไม่ครบทุกช่องปุ่มจะทำการ disable ไม่สามารถใช้งานปุ่มได้
